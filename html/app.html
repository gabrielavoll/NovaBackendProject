Welcome to NovaBackendProject

possible urls to hit:

  • /phase1
    accepts a GET request, requires params: country_code, account_id, routing_number, file_ext, tags
    initializes a data transfer; saves a pending transfer record to db
  • /phase2
    accepts a POST request, requires both a cookie token and json body that contains: country_code, account_id, routing_number, first_name, last_name, credit_limit, credit_score
    uploads the file send POST body, if it follows the specifications sent up in /phase1, requires matching country_code, account_id, routing_number and if the cookie token matches the /phase1 record
  • /data/:id
    accepts GET request, displays data from successful /phase1 & /phase2 completion
